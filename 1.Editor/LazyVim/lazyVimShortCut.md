# LazyVim

```
<leader> space

退出插入模式      jj  -i

窗口间光标跳动    CTRL + hjkl 
改变窗口的大小    CTRL + 上下左右小键盘 

交换两行 | 改变当前行位置 ALT + jk | <LEADER> ur

取消搜索高亮      ESC

保存文件          CTRL s

文件搜索          <LEADER><SPACE>
最近使用文件搜索          <LEADER> fr
从文件搜索退出 取消文件搜索 CTRL c

向下添加一行注释 gco
向上添加一行注释 gcO

打开|关闭目录(explorer) <LEADER> e
快速格式化       <LEADER> cf
关闭所有         <LEADER> qq

查看LazyVim改动情况 <LEADER> L

打开窗口并进行窗口操作 <LEADER> w
向下拆分窗口            <LEADER> -
向右拆分窗口            <LEADER> |
删除当前窗口            <LEADER> wd

TAB :
    最后一个TAB : <LEADER> <TAB>l
    关闭其他TAB : <LEADER> <TAB>0
    新建一个TAB : <LEADER> <TAB><TAB>
    前一个TAB : <LEADER> <TAB>[
    后一个TAB : <LEADER> <TAB>]
    关闭一个TAB: <LEADER><TAB>d
```
